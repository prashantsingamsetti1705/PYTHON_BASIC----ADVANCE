				==========================================================
							Nested OR Inner Loops
				==========================================================
=>The Process of Defining One Loop inside of another Loop is called Nested OR Inner Loops.
=>The Execution Process of Inner Loops is that "For Every Value of Outer Loop, Inner Loop Repeated for Finite 
    Number of Times until Inner Loop Test Condition becomes False"
=>We can Nested OR Inner Loops in 4 Ways.
--------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax-1:  for loop in for loop
--------------------------------------------------------------------------------------------------------------------------------------------------------------
		for  Varname1 in Iterable-Object1:  # Outer Loop
			---------------------------------------
			for Varname2 in  Iterable-Object2:  # Inner Loop
				----------------------------------------
				----------------------------------------
			else:
				-----------------------------------------
		else:
			------------------------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax-2:  while loop in while loop
--------------------------------------------------------------------------------------------------------------------------------------------------------------
		---------------------------
		while (Test Cond1): # Outer Loop
			-------------------------
			-------------------------
			while(Test Cond2): # inner Loop
				-----------------------
				-----------------------
			else:
				------------------------
				------------------------
		else:
			----------------------------------
			----------------------------------

--------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax-3:  for loop in while loop
--------------------------------------------------------------------------------------------------------------------------------------------------------------
		---------------------------
		while (Test Cond1): # Outer Loop
			-------------------------
			-------------------------
			for Varname2 in  Iterable-Object2:  # Inner Loop
				----------------------------------------
				----------------------------------------
			else:
				-----------------------------------------
		else:
			----------------------------------
			----------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax-4:  while loop in for loop
--------------------------------------------------------------------------------------------------------------------------------------------------------------
		for  Varname1 in Iterable-Object1:  # Outer Loop
			---------------------------------------
			while(Test Cond2): # inner Loop
				-----------------------
				-----------------------
			else:
				------------------------
				------------------------
		else:
			------------------------------------------------
